Fliplt微软复古翻页时钟屏保

Fliplt微软复古翻页时钟屏保

  • 翻转时钟屏幕保护程序。

  • 灵感来自 Fliqlo。

  • Windows 上的 Fliqlo 停止使用最近（2015 年 12 月？）Flash 更新，这促使了这个项目。

  • 这个项目不使用 Flash。

要在不使用 Visual Studio 构建的情况下进行安装，请将发布页面上的 .scr 文件复制到：

  • C:\Windows\SysWOW64（在 64 位 Windows 上）
  • C:\Windows\System32 （在 32 位 Windows 上）

最后即可在电脑屏幕保护程序处进行配置！
